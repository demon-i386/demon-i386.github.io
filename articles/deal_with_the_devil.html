Gas

                                used for preventing issues such as infinite code running and network spam is a fee for using computational power

                                1 gwei = 0,00000001 ETH

                                Total gas cost = Operation cost * Gas price (gwei)
                                Transaction fee = Operation cost * Gas price (gwei) * Ethereum price in local currency

                                Miners profit from gas and choose priority transations based on transaction fee, anyone 
                                can speed up the queue paying more for the computation.
                                Mining is the process of creating, verifying, publishing or propagating a block of transactions 
                                to the blockchain.


                                Example:
                                Obs: Simple transaction, there is no contract code being executed, so there's arent Operation cost fee.
                                
                                Transaction fee = Gas limit * (Base fee + Tip)

                                - Gas limit: The maximum amount that is willing to be used in a transaction, refunded if unused. (minimum: 21000 gwei)

                                - Base free: Minimum amount of gwei to include the transaction on blockchain, dictated by the congestion level
                                of the network. (dynamic)

                                - Tip: Miner priority fee, users pay to the miners in order to prioritize their transactions, a large payout 
                                ensures prioritization of transaction confirmation, accelerating your process over the network. (arbitrary)

                                
                                Prices used in this example can be outdated
                                
                                Alice pays Bob 1 ETH. 
                                The values of the fee is: the gas limit (minimum of 21,000 gwei), the base fee of 33 gwei and a tip of 20 gwei.

                                Using the transaction fee formula (Transaction fee = Gas limit * (Base fee + Tip)) we can discover the total cost
                                to complete this transfer.

                                Transaction fee = 21,000 * (33 + 20)
                                Transaction free = 11,130.00 gwei (0.001618 ETH)

                                When Alice completes the transaction, 1.001618 ETH will be deducted her account. 
                                Bob will be credited 1.0000 ETH. 
                                Miner will receive the tip of 0.0000000291 ETH (20 gwei). 
                                Base fee of 0.0000000480 ETH (33 gwei) is burned.
